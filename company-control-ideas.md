# 自社コントロール改善案

今の自社機能は、会社を作って市場に並べるところで止まっています。これだと「自分の会社を持った」という気持ちは出ますが、次に何をしたら面白いのかが薄いです。

目指したいのは、プレイヤーが自社を「ただの銘柄」ではなく「自分の物語を作る装置」として触れる状態です。自社のニュースを出し、勝負を仕掛け、失敗して炎上し、それでも立て直して、他プレイヤーがそれを見て買ったり売ったりする。そうなると市場に会話と事件が生まれます。

このドキュメントでは、GitHub Pages + Firebase Realtime Database の無料構成でも実装しやすい案から、将来的に入れると一気に化ける案まで、ワクワク重視でまとめます。

## 方向性

自社コントロールは、ただステータスを上げるだけだと作業になります。面白くするには、ボタンを押した瞬間に「市場が少しざわつく」必要があります。

そこで、自社機能は次の4本柱にします。

| 柱 | プレイヤー体験 | 例 |
|---|---|---|
| 経営判断 | 会社をどう育てるか選ぶ | 研究開発、採用、広告、品質改善 |
| 市場演出 | 自社ニュースを市場に流す | 新商品発表、決算発表、提携発表 |
| リスクと報酬 | 成功も失敗もある勝負をする | 大型投資、炎上、リコール、復活劇 |
| 他者との関係 | 他プレイヤーを巻き込む | 株主還元、買収防衛、共同プロジェクト |

## 最高に入れたい機能セット

最初から全部は不要ですが、最終的にはこの形が一番おいしいです。

| 機能 | 面白さ | 実装難度 |
|---|---|---|
| 社長アクション | 自分の会社を毎日動かしている感覚が出る | 低 |
| 自社ニュース発表 | 市場に自分の名前が出て目立つ | 低 |
| 成功・失敗判定 | 押すだけではなく勝負になる | 中 |
| 決算イベント | 会社に周期的な山場ができる | 中 |
| 株主人気 | 他人に買われるほど会社が強くなる | 中 |
| ライバル企業指定 | プレイヤー間の物語が生まれる | 中 |
| 大型プロジェクト | 数日かけて育てる目標ができる | 中 |
| 買収・提携 | 市場全体に事件が起きる | 高 |

## 自社ステータス

自社には、ただの数値ではなく「会社の個性」が見えるステータスを持たせます。

| ステータス | 意味 | 高いと起きること |
|---|---|---|
| 技術力 | 商品やサービスの強さ | 新商品発表が成功しやすい |
| ブランド | 人気と知名度 | 広告やニュースの株価効果が大きい |
| 信頼 | 不祥事への耐性 | 悪材料イベントで暴落しにくい |
| 財務 | 会社の体力 | 大型投資、配当、自社株買いができる |
| 社員熱量 | 勢いと実行力 | プロジェクト成功率が上がる |
| 注目度 | 市場での話題性 | ニュースが他プレイヤーの目に入りやすい |

ポイントは、最強ステータスを1つにしないことです。技術力だけ高い会社、ブランドだけ強い会社、信頼だけ異常に高い会社など、プレイヤーごとに会社の性格が分かれると楽しくなります。

## 社長アクション

自社画面には「社長アクション」を並べます。これは株を買う・売るとは別に、会社そのものに手を入れるボタンです。

| アクション | 費用 | 成功時 | 失敗時 |
|---|---:|---|---|
| 新商品を発表 | 150,000円 | 自社株が上昇、技術力とブランド上昇 | 期待外れで小下落 |
| 広告キャンペーン | 80,000円 | 注目度上昇、短時間だけ買われやすい | 効果なし |
| 優秀な社員を採用 | 120,000円 | 社員熱量上昇、次の施策成功率上昇 | 採用コストだけ失う |
| 品質改善 | 90,000円 | 信頼上昇、悪材料耐性アップ | 即効性なし |
| コスト削減 | 50,000円 | 財務上昇 | 信頼または社員熱量が下がる |
| 強気の決算発表 | 無料 | 成功なら大きく上昇 | 失敗なら大きく下落 |
| 謝罪会見 | 30,000円 | 炎上・悪材料の影響を軽減 | 注目度が下がる |
| 自社株買い | 200,000円 | 自社株を下支え | 財務が減る |
| 配当発表 | 150,000円 | 株主人気上昇 | 財務が減る |
| 秘密プロジェクト開始 | 300,000円 | 数分後に大イベント | 失敗すると市場が失望 |

この中で特に面白いのは「強気の決算発表」と「秘密プロジェクト開始」です。プレイヤーが自分で爆弾を置く感じがあり、成功しても失敗しても話題になります。

## 成功率の考え方

アクションは必ず成功ではなく、会社ステータスで成功率が変わるとゲームになります。

例として、新商品発表なら次のようにします。

```js
successRate = 40
  + technology * 2
  + employeeMorale * 1
  + brand * 0.5;
```

成功率は最低10%、最高90%くらいに丸めます。これで、弱い会社でもワンチャンがあり、強い会社でも油断できません。

## 自社ニュース

社長アクションを実行したら、市場のイベント通知に自社ニュースを流します。

例:

| アクション | ニュース文 |
|---|---|
| 新商品を発表 | 「〇〇社が新型サービスを発表。市場は成長期待を強めています。」 |
| 失敗した新商品 | 「〇〇社の新商品に厳しい評価。投資家の失望売りが出ています。」 |
| 広告成功 | 「〇〇社の広告キャンペーンが話題化。注目度が急上昇しています。」 |
| 秘密プロジェクト成功 | 「〇〇社が謎の大型プロジェクトを公開。株価は大きく反応。」 |
| 謝罪会見 | 「〇〇社が緊急会見。信頼回復に向けた対応を発表しました。」 |

ここがかなり大事です。自分の行動が市場ニュースとして全員に見えると、ただのボタンが「事件」になります。

## 株主人気

他プレイヤーが自社株を持っているほど、自社に「株主人気」が貯まるようにします。

| 状態 | 効果 |
|---|---|
| 他人が自社株を買う | 株主人気 +1 |
| 他人が自社株を大量保有 | ニュース効果が少し上がる |
| 他人が一斉に売る | 株主人気が下がり、信頼も少し下がる |
| 株主人気が高い | 配当・決算・新商品発表が盛り上がりやすい |

これが入ると、自社を強くするには「自分で買う」だけでなく「他人に買ってもらう」ことが重要になります。自然に宣伝したくなるし、他プレイヤーの会社を見る理由もできます。

## ライバル企業

プレイヤーは1社だけライバル企業を指定できます。

ライバルを指定すると、相手のニュースが自分の画面で強調されます。また、いくつかのアクションにライバル効果を付けられます。

| アクション | 効果 |
|---|---|
| 比較広告 | 成功すると自社ブランド上昇、ライバル注目度低下 |
| 技術競争 | 技術力が高い方に好材料イベント |
| シェア争い | 同じセクターの会社同士で株価が逆方向に動く |
| 共同規格 | ライバルと両方上がるが、相手も得をする |

これはかなり盛り上がります。実害を大きくしすぎると荒れるので、最初は「演出と小さな補正」くらいが安全です。

## 大型プロジェクト

数分から数時間かけて進む「大型プロジェクト」を入れると、短期売買だけではない目標ができます。

| プロジェクト | 必要条件 | 成功時 | 失敗時 |
|---|---|---|---|
| 次世代AI開発 | 技術力 5以上 | 技術力大幅上昇、株価上昇 | 財務低下 |
| 全国広告展開 | ブランド 5以上 | 注目度と株主人気上昇 | 費用だけ消える |
| 海外進出 | 財務 5以上 | 成長力上昇 | 信頼低下 |
| 品質改革 | 信頼 3以上 | 悪材料耐性大幅上昇 | 目立った効果なし |
| 夢の新商品 | 技術力・社員熱量が必要 | 大成功なら急騰 | 失敗なら急落 |

大型プロジェクトは、開始した時点で市場に「〇〇社が大型プロジェクトを開始」と出します。そして完了時にもニュースを出します。待っている間に他プレイヤーが先回りして買うか、疑って売るかを選べます。

## 決算システム

自社には1日1回または数時間に1回、決算を出せるようにします。

決算には3種類あります。

| 決算タイプ | 内容 | 向いている会社 |
|---|---|---|
| 堅実決算 | 小さく上がりやすい | 信頼が高い会社 |
| 成長決算 | 成功すれば大きく上がる | 技術力・ブランドが高い会社 |
| サプライズ決算 | 大成功か大失敗 | 注目度が高い会社 |

決算前には「決算予定」表示を出すと面白くなります。市場が待ち構える感じが出るからです。

## 炎上と復活

悪いイベントも、ただ下がるだけではなく、立て直しができると楽しいです。

| 状態 | 起きること | 回復方法 |
|---|---|---|
| 炎上 | ブランドと信頼が下がる | 謝罪会見、品質改善 |
| 人材流出 | 社員熱量が下がる | 採用、福利厚生 |
| 資金難 | 大型アクション不可 | 増資、コスト削減 |
| 製品不良 | 新商品系アクション成功率低下 | 品質改革 |

復活劇があると、株価が下がった時もゲームが続きます。「もう終わり」ではなく「ここから立て直す」ができるのが大事です。

## 社風カード

会社設立時、または後から1つだけ「社風」を選べるようにします。これは会社の個性を決めるパッシブ能力です。

| 社風 | 効果 |
|---|---|
| 技術者集団 | 技術力が伸びやすいが広告効果は弱い |
| カリスマ企業 | ブランドが伸びやすいが炎上に弱い |
| 堅実経営 | 信頼と財務が伸びやすいが急騰しにくい |
| 勝負師 | 大成功しやすいが大失敗も増える |
| ホワイト企業 | 社員熱量が下がりにくい |
| 炎上商法 | 注目度が高いが信頼が上がりにくい |

これは絶対に面白いです。プレイヤーが自分の会社にキャラ付けできます。

## 称号と実績

自社の行動に応じて称号がつくと、数字以上に楽しくなります。

| 条件 | 称号 |
|---|---|
| 新商品を3回成功 | 革新企業 |
| 信頼20達成 | 鉄壁企業 |
| 炎上から株価回復 | 復活の社長 |
| 株主人気30達成 | 市場の人気者 |
| 大型プロジェクト成功 | 次世代の覇者 |
| 決算で大失敗 | 伝説の下方修正 |

称号は市場のカードや企業詳細に表示すると、他人から見ても楽しいです。

## プレイヤー間イベント

本当に盛り上げるなら、他プレイヤーとの関係を少しだけ入れます。

| イベント | 内容 |
|---|---|
| 共同開発 | 2社の技術力を使って成功判定。両社に好材料 |
| 業務提携 | 片方のブランド、片方の技術力を組み合わせる |
| 敵対的買収の噂 | 相手株を持っていると発生。市場がざわつく |
| 株主総会 | 他プレイヤーの保有比率でボーナスが変わる |
| セクター戦争 | 同じ分野の会社が一斉に比較される |

最初は実際の交渉 UI なしで、「提携申請ボタン」と「承認ボタン」だけでも十分です。

## 自社画面の理想

会社を持っている時の「会社設立」タブは、名前を「自社経営」に変えたいです。

画面には以下を置きます。

| エリア | 内容 |
|---|---|
| 社長室 | 会社名、現在株価、今日の評価、称号 |
| 経営メーター | 技術力、ブランド、信頼、財務、社員熱量、注目度 |
| 今日の一手 | 主要アクションを3つだけ大きく表示 |
| 大型プロジェクト | 進行中プロジェクト、残り時間、成功予想 |
| 市場の反応 | 最新ニュース、株主人気、他人の保有状況 |
| 履歴 | 最近の経営判断と結果 |

「今日の一手」を3つだけ大きくするのがポイントです。選択肢が多すぎると迷うので、普段は3つ、詳細を開くと全部見えるくらいがよいです。

## 実装しやすい順番

まずは、この順番が一番よさそうです。

1. 自社ステータスを追加する
2. 自社経営画面を作る
3. 社長アクションを3つ入れる
4. アクション結果を市場ニュースに出す
5. 自社株価にステータス補正を入れる
6. 成功・失敗判定を入れる
7. 大型プロジェクトを入れる
8. 決算システムを入れる
9. 株主人気を入れる
10. ライバル・提携を入れる

## 最初のバージョン案

最初の実装は、このくらいがちょうどいいです。

| 項目 | 内容 |
|---|---|
| ステータス | 技術力、ブランド、信頼、財務、社員熱量、注目度 |
| アクション | 新商品発表、広告キャンペーン、品質改善、コスト削減、強気の決算発表 |
| 成功判定 | ステータスに応じて10%から90% |
| ニュース | 成功・失敗を全員の市場通知に表示 |
| クールダウン | 5分に1回 |
| 株価反映 | 成功で短期上昇、失敗で短期下落、ステータスで自然変動に小補正 |

この時点で、かなりゲームの顔が変わります。

## さらに尖らせる案

もっと遊びを強くするなら、少し変なアクションも入れたいです。

| アクション | 効果 |
|---|---|
| 謎のティザーサイト公開 | 数分間だけ注目度が上がる。後で成功判定 |
| 社長が強気発言 | 成功ならブランド上昇、失敗なら信頼低下 |
| 緊急ライブ配信 | 炎上中なら回復チャンス、平常時なら注目度上昇 |
| 株主優待を発表 | 株主人気上昇、財務低下 |
| 社名変更 | ブランドをリセット気味に再抽選。話題性は大きい |
| 一発逆転プロジェクト | 成功率低め、大成功時だけ急騰 |

こういう少しドラマのあるボタンがあると、友達同士で「押せ押せ」「やめとけ」が起きます。

## バランスの考え方

自社が強くなりすぎると、全員が自社だけ買えばよくなってしまいます。なので、強化には必ず代償をつけます。

| 強い効果 | 代償 |
|---|---|
| 株価を上げる | 財務が減る、失敗リスクがある |
| 成功率を上げる | 時間がかかる |
| 注目度を上げる | 炎上時の下落も大きくなる |
| 財務を増やす | ブランドや社員熱量が下がる |
| 信頼を上げる | 短期的な株価上昇は弱い |

「どれも正解だけど、全部は取れない」状態が理想です。

## まとめ

一番大事なのは、自社操作を「数字を上げる作業」ではなく「市場に事件を起こす遊び」にすることです。

最初に作るなら、次の5つを入れるのがおすすめです。

1. 自社ステータス
2. 社長アクション
3. 成功・失敗判定
4. 自社ニュース通知
5. 大型プロジェクト

これで、自社はかなり面白くなります。プレイヤーはただ株価を見るだけでなく、「今日は何を発表するか」「勝負するか守るか」「他人に買ってもらえる会社にするか」を考えるようになります。
